Configuración

Categoría: {{ page.category }}

Sincronización
  • Para realizar la sincronización Summar precisará de un usuario y su web access key (no el password).
  • Por favor, procure que la web access key nunca caduque ya que sino el sistema de sincronización se detendría.
NOTA: Consulte este enlace para saber cómo configurar un web access key.

General
  • Busque y ejecute "TAXOR Cloud Configuración".
  • Haga click en "Obtener empresas". Le mostrará por pantalla las empresas que se sincronizarán con Taxor Cloud.
Usuarios
  • El sistema asignará automáticamente los permisos requeridos a todos los usuarios existentes. Si desea que algún usuario no acceda a la funcionalidad proporcionada por la app deberá quitarle el permiso. Igualmente, cuando cree nuevos usuarios, si desea que acceda a la funcionalidad proporcionada por la app deberá asignarle el permiso. El nombre del permiso (Role ID) es TAX-CONECTOR
Para cada empresa
  • Diríjase a cada una de las empresas que se están sincronizando y busque y ejecute "TAXOR Cloud Configuración".
  • En la pestaña General podrá activar o desactivar las funcionalidades de Retenciones e IVA.
NOTA: La funcionalidad de Retenciones estará desactivada para impedir que entre en conflicto con alguna otra funcionalidad similar que pueda tener instalada.

Retenciones
  • Informe las cuentas contables a utilizar en cada uno de los campos.
Inmuebles
Puede informar de los inmuebles arrendados y, si lo desea, asignarles una dimensión y un valor de dimensión. De esta forma si el proveedor tiene dicha dimensión y valor asignados, el sistema vinculará automáticamente la retención con el inmueble. En caso contrario deberá indicarlo manualmente.
NOTA: Deberá hacer coincidir el código con su correspondiente Inmueble en Taxor Cloud, ya que los inmuebles no se sincronizan.

IVA
  • Informe cómo va a realizar el control de operaciones de intragrupo. Existen 2 opciones:
    1. Mediante intercompany. El sistema determinará si el campo "Código socio IC" está informado en la ficha de los clientes o proveedores y si el socio es otra empresa del grupo.
    2. Mediante dimensión: El sistema determinará qué clientes o proveedores tienen esa dimensión y valor de dimensión informada, en cuyo caso indicará que se trata de una empresa del grupo.
Actividades
Puede informar de las actividades y, si lo desea, asignarles una dimensión y un valor de dimensión. De esta forma si el cliente o proveedor tiene dicha dimensión y valor asignados, el sistema vinculará automáticamente la operacion con la actividad. En caso contrario deberá indicarlo manualmente.
NOTA: Deberá hacer coincidir el código con su correspondiente Actividad en Taxor Cloud, ya que las actividades no se sincronizan.

Legalización de libros
  • Al instalar la extensión el sistema ya realiza una configuración previa, pero es probable que algunos informes no puedan ser establecidos apropiadamente. Por favor, revísela.
