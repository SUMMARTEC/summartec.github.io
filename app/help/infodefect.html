Información por defecto


La información definida por defecto en la ficha de un tercero, como un cliente, proveedor o empleado, se trasladará automáticamente donde proceda al realizar alguna transacción.
Si bien es cierto que, aún y así, podrá cambiarla antes de registrar. I.V.A.
  • Código socio IC
    • Se ha añadido un botón de asistencia (identificado mediante puntos suspensivos) que, en el caso que el socio sea otra empresa existente en Business Central, le permitirá identificarla y el sistema la auto-configurará automáticamente para usted.
  • Taxor Intragrupo
    • Este campo le indicará, en el caso de que el código de socio IC esté informado, si el socio ha sido identificiado como empresa del grupo.
    Si así se ha indicado en la configuración, este campo se trasladará automáticamente a los documentos de compra y venta correspondientes.

I.R.P.F.
  • Modelo IRPF
    • Debido a que las claves y subclaves del modelo 193 son distintas al resto, indique en este campo el modelo dseado escogiéndolo de la lista.
  • Clave IRPF
    • Indique en este campo la clave de IRPF deseada escogiéndola de la lista. Fíjese que este campo es dependiente del anterior.
  • Subclave IRPF
    • Indique en este campo la subclave de IRPF deseada escogiéndola de la lista. Fíjese que este campo es dependiente del anterior.
  • % Retención
    • Indique en este campo el % de retención a aplicar.
  • Arrendatario / Arrendador
    • Indique en este si el tercero coincide con alguna de estas figuras. Esta información se trasladará a ciertas declaraciones en las que se solicita explícitamente.
  • Inmueble por defecto
    • Indique en este campo el inmueble que se utilizará por defecto al registrar alguna transacción. Si la transacción final incluye algún inmueble, se marcará igualmente como Arrendatario / Arrendador anque el campo anterior esté desactivado.
  • Importe Retenciones
    • Este campo indica el importe total de las retenciones practicadas al tercero en cuestión. Navegue para ver el detalle.
nota: si el documento incluye una cuenta de activo (grupo 2) el sistema lo marcará como bien de inversión
