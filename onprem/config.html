Configuración


El objetivo es que la infraestructura de Summar en Azure pueda conectarse a su SQL Server para obtener los datos de su ERP necesarios para Taxor Cloud.
Existen varias opciones para realizar la sincronización. Escoja la que más le convenga.

Extensión
  • Si su versión de Business Central es igual o superior a la versión 17 (octubre 2020) puede optar por instalar la extensión para Taxor Cloud.
  • Será necesario licenciar el Add-on de Taxor a través de su partner.
NOTA: Consulte este enlace para saber más.

Data Gateway
  • Microsoft On-premise Data Gateway va a permitir que se realice una conexión totalmente segura y exclusivamente desde su infraestructura informática y la infraestructura de Summar en Azure. Lo mejor es que no hay que tocar nada en su infraestructura, ni routers ni firewalls. Puede obtener información en ¿Qué es una puerta de enlace de datos local?
  • Para proceder con la instalación vea este enlace Instalar una puerta de enlace de datos local desde el que podrá saber los requisitos y consideraciones, además de descargar el ejecutable de instalación.
  • Es indispensable que la instalación se realice en un ordenador o servidor con acceso a Internet y al servidor SQL y que no se detenga durante las noches. Si es un servidor, no puede ser controlador de dominio.
  • Al instalar se solicitará una cuenta de correo electrónico de Office 365 y el nombre de la puerta de enlace. Utilice para ello la información suministrada por Summar con el objetivo de que ésta puerta de enlace esté exclusivamente conectada con la infraestructura de Summar en Azure.
Firewall
  • Esta opción no requiere de ninguna instalación de ningún software en su infraestructura informática. Eso sí, deberá crear alguna regla en su Firewall y en caso de que posea infraestructura propietaria quizá deba tocar también su router. Puede y debe limitar las IPs que pueden acceder al puerto SQL. Para este caso concreto, debe saber que la infraestructura en Azure de Summar se halla en West Europe, utilice para ello la columna "Managed connectors IP" con las IPS indicadas en Limits and configuration - Azure
  • Si su infraestructura informática se encuentra en Azure va a resultar mucho más fácil, tan sólo debe configurar el Tag AzureConnectors - Azure
En los dos casos anteriores
  • Deberá instalar en la BBDD del ERP unos “Stored Procedures” suministrados por Summar. Estos procedimientos serán utilizados exclusivamente para obtener (read only) la información precisa desde el ERP hacia TAXOR Cloud.
  • Finalmente, deberá suministrar a Summar un usuario y contraseña con acceso a SQL Server y permisos para ejecutar los “Stored procedures” y acceder a los datos del ERP. Por favor, procure que la contraseña nunca caduque ya que sino el sistema de sincronización se detendría.
  • Para su información, la sincronización se realizará siempre durante las noches. Si representa un problema o existe algún horario preferido háganoslo saber.
